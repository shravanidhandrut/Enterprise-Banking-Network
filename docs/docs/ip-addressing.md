# 📡 IP Addressing Plan

This document provides a detailed breakdown of the IP addressing scheme used in the **Enterprise Banking Network**.

---

## 🔹 Overview

* **Base LAN Network:** `192.168.10.0/24`
* **Subnetting Scheme:** `/26` (255.255.255.192)
* **Hosts per Subnet:** 62 usable hosts
* **Total VLANs:** 12 (one per department)

---
